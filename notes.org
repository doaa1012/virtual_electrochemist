** Objective
- Find suitable descriptors for cyclic voltammograms

** How
- Create a reference benchmark of narrative descriptions of experts
  describing existing CV/LSV curves

** Details
- Which kind of metadata is necessary for a descriptive interpretation
  of a CV curve for an expert?
  + Intended reaction (HER/Ammonia oxidation)
  + Catalyst composition (elemental concentrations in overall cell in contact)
  + Electrode material (e.g. something on Ni, C, etc.)
  + Catalyst morphology (optional)
  + Electrolyte: composition, e.g. 1M KOH; more complicated: pH value;
    bubbling (e.g. Ar, O, air)
  + Environment of experiment (atmosphere, temperature)
  + Reference electrode
  + Scan rate, potential range (explicit)
  + IR correction/compensation yes/no and how
  + publication reference
  + Any other additional information
  + Normalization procedure w.r.t area (current density)
